#  Taqui Science Academy Admission Website

A responsive and modern academy admission website developed using HTML, CSS, Bootstrap, JavaScript, and Node.js. The website provides information about the institution and allows students to navigate through the admission process easily.

---

##  Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and user-friendly interface
- Home page with school information
- Admission application page
- Login page
- Navigation menu
- Bootstrap-powered UI components
- Fast and lightweight performance

---

##  Technologies Used

- HTML
- CSS
- Bootstrap 5
- JavaScript
- Node.js
- Express.js

##  Installation

###  Clone the Repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### Navigate to the Project Directory

```bash
cd your-repository-name
```

###  Install Dependencies

```bash
npm install
```

###  Start the Server

```bash
node server.js
```

The application will run at:

```text
http://localhost:3000
```
